# Действия

Действия описываем следующим образом:

- Переменная или другой получатель
- Действие
- Функция или значение

let, const — тип данных, хранящийся в переменной
= — записываем
function — функция
addEventListener — Добавляем отслеживание событий по триггеру

add* - отображаем
remove* - убираем
